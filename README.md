# BambaERP Android App

Application Android WebView pour https://bambacorp.mesh.vitib.de/

## Comment générer l'APK (sans Android Studio)

### Méthode GitHub Actions (recommandée)

1. Crée un compte sur [github.com](https://github.com) si tu n'en as pas
2. Crée un nouveau dépôt public nommé `BambaERP`
3. Upload tous les fichiers de ce projet dans le dépôt
4. Va dans l'onglet **Actions** → le build démarre automatiquement
5. Attends ~3-5 minutes → clique sur le build terminé
6. Télécharge l'artifact **BambaERP-debug** → tu obtiens l'APK

### Installation sur Android

1. Transfère le fichier `.apk` sur le téléphone
2. Si bloqué : Paramètres → Sécurité → Autoriser les sources inconnues
3. Ouvre le fichier `.apk` et installe
